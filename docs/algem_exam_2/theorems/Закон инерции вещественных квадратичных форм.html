Закон инерции вещественных квадратичных форм

Определение: Сигнатура формы

Из-за отсутствия корня из отрицательного числа, над $\mathbb{R}$ форму можно привести к лишь к виду: $$f \sim y_{1}^{2} + y_{2}^{2} + \dots + y_{k}^{2} - y_{k+1}^{2} - \dots - y_{k+s}^{2}$$ Сигнатурой формы называется пара $(r, k - s)$, где $r$ - ранг формы, $k$ - кол-во $+$, $s$ - кол-во $-$. $r(f) = k + s$

Теорема: Закон инерции квадратичных форм

Формулировка:

Число положительных и отрицательных слагаемых в каноническом виде квадратичной формы инвариантно относительно способа приведения.

Д-во:


Пусть форма $h = h(x_1, \ldots, x_n)$ приводится к двум каноническим видам:
$$f = \sum_{i=1}^k t_i y_i^2 - \sum_{j=k+1}^{k+l} t_j y_j^2 \qquad g = \sum_{i=1}^p s_i z_i^2 - \sum_{j=p+1}^{p+q} s_j z_j^2$$
где $r(f) = k+l = p + q = r$ ($t_{i}, s_{i} > 0$).

Достаточно доказать, что $k = l$, так как тогда и $s = t$.
От противного: Пусть $k < p$

Пусть $f$ получено невырожденной заменой $T_{f}$, а $g$ - $T_{g}$. Тогда замена $T_{f}^{-1} \circ T_{g}$ приводит форму $f$ к виду $g$. Рассмотрим эту замену:
$$
\begin{cases}
y_1 = d_{11}z_1 + d_{12}z_2 + \cdots + d_{1n}z_n, \\
y_2 = d_{21}z_1 + d_{22}z_2 + \cdots + d_{2n}z_n, \\
\vdots \\
y_n = d_{n1}z_1 + d_{n2}z_2 + \cdots + d_{nn}z_n,
\end{cases}
$$
и подставим в $f$ вместо $y_1, \ldots, y_n$. Получаем тождество:
$$\sum_{i=1}^k t_i y_i^2 - \sum_{j=k+1}^{k+l} t_j y_j^2 = \sum_{i=1}^p s_i z_i^2 - \sum_{j=p+1}^{p+q} s_j z_j^2$$
Перегруппируем по знаку:
$$\sum_{i=1}^k t_i y_i^2 + \sum_{j=p+1}^{p+q} s_j z_j^2 = \sum_{i=1}^p s_i z_i^2 + \sum_{j=k+1}^{k+l} t_j y_j^2 \qquad (1)$$
Теперь рассмотрим систему по переменным в левой части:
$$y_1 = y_2 = \cdots = y_k = z_{p+1} = z_{p+2} = \cdots = z_n = 0$$
Та же система в развёрнутом виде:
$$\begin{cases}
d_{11}z_1 + d_{12}z_2 + \cdots + d_{1n}z_n = 0 \\
d_{21}z_1 + d_{22}z_2 + \cdots + d_{2n}z_n = 0 \\
\vdots \\
d_{k1}z_1 + d_{k2}z_2 + \cdots + d_{kn}z_n = 0 \\
z_{p+1} = 0 \\
z_{p+2} = 0 \\
\vdots \\
z_n = 0
\end{cases}$$
Так как $k < p$, то число уравнений $k + n - p = n + (k - p) < n$. Следовательно, система имеет ненулевое решение $(u_1, u_2, \dots, u_p, 0, \dots, 0)$. Подставляя это решение в $(1)$, получаем:
$$
0 = \sum_{i=1}^p s_i u_i^2 + \sum_{j=k+1}^{k+l} t_j y_j^2
$$
что невозможно, поскольку правая часть положительна (решение ненулевое), противоречие.

Следовательно, $k \geq p$. Аналогично доказывается $k \leq p$, а значит ${} k = p {}$
$\square$
