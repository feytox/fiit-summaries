Кольцо многочленов

* Определение: Кольцо


Напомним, что кольцо - множество $K$ с бинарными операциями $+$ и $\cdot$ со следующими свойствами для любых $a, b, c \in K$:

  1. $a + b = b + a$

  2. $a + (b + c) = (a + b) + c$

  3. $\exists{0 \in K}\mathpunct{:}~~ a + 0 = a$

  4. $\exists{(-a)}\mathpunct{:}~~ a + (-a) = 0$

  5. $(a \cdot b) \cdot c = a \cdot (b \cdot c)$

  6. $a \cdot (b + c) = (a \cdot b) + (a \cdot c)$

  7. $(b + c) \cdot a = (b \cdot a) + (c \cdot a)$

Определение: Кольцо многочленов

Кольцом многочленов называется кольцо бесконечных строк с конечным числом ненулевых элементов в каждой строке с операциями: - Сложение: $$(a_{0}, a_{1}, \dots, a_{n}, 0, \dots) + (b_{0}, b_{1}, \dots, b_{m}, 0, \dots) = (a_{0} + b_{0}, a_{1} + b_{1}, \dots)$$ - Умножение: $$(a_{0}, a_{1}, \dots, a_{n}, 0, \dots) \cdot (b_{0}, b_{1}, \dots, b_{m}, 0, \dots) = (a_{0}b_{0}, a_{1}b_{0} + a_{0}b_{1}, \dots)$$

Определение: Стандартный вид многочлена

Стандартным видом многочлена называется запись вида: $$f(x) = a_{0} + a_{1}x + a_{2}x^{2} + \dots + a_{n}x^{n} = \sum_{k=0}^{n} a_{k}x^{k}$$ Каждое слагаемое называется одночленом (или мономом)

Определение: Степень многочлена


Одночлен ненулевого многочлена с наивысшей степенью $x$ называется старшим членом

Показатель степени в старшем члене называется степенью многочлена.
Обозначение: $\deg f$
По определению, $\deg 0 = -\infty$
