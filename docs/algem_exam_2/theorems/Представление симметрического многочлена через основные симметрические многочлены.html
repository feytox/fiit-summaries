Представление симметрического многочлена через основные симметрические многочлены

Основная теорема о симметрических многочленах

$f(x_{1}, x_{2},\dots, x_{n})$ - симметрический многочлен, тогда $f$ - многочлен от элементарных симметрических многочленов.

Лемма

Формулировка:


$$mdeg(f_{1}f_{2}) =mdeg(f_{1}) + mdeg(f_{2})$$
$$lead(f_{1}f_{2}) = lead(f_{1}) \cdot lead(f_{2})$$
- $mdeg⁡(f)$ — мультистепень старшего монома $f$,

  • $lead⁡(f)$ — старший моном $f$ (моном максимальной степени).
    Д-во: $\dagger \dagger$spoiler-markdown
    $$f_{1} = A_{1}x_{1}^{a_{1}}x_{2}^{a_{2}}\dots x_{n}^{a_{n}} + \bar{f_{1}}$$
    $$f_{2} = B_{1}x_{1}^{b_{1}}x_{2}^{b_{2}}\dots x_{n}^{b_{n}} + \bar{f_{2}}$$
    $f_{1} \cdot f_{2} = A_{1}B_{1}x_{1}^{a_{1} + b_{1}}x_{2}^{a_{2} + b_{2}}\dots x_{n}^{a_{n} + b_{n}} +\dots$

далее по индукции.

Д-во(теоремы):


Пусть $\mathcal{F} =$ ${$симметрические многочлены, которые не выражаются через$~\sigma_{k} }$.
Предположим $\mathcal{F} \neq \varnothing$. Пусть $f \in \mathcal{F}$ с наименьшей мультистепенью.
$mdeg{f} = (a_{1}, a_{2}, \dots, a_{n})$ и $a_{1} \geq a_{2} \geq \dots \geq a_{n}$ (из симметричности)
Рассмотрим старший одночлен $f$ и выразим его через элементарные симметрические:
$$Ax_{1}^{a_{1}}x_{2}^{a_{2}} \dots x_{n}^{a_{n}} = A \sigma_{1}^{a_{1}} \sigma_{2}^{a_{2}} \dots \sigma_{n}^{a_{n}} + \dots$$
$mdeg{~A \sigma_{1}^{a_{1}} \sigma_{2}^{a_{2}} \dots \sigma_{n}^{a_{n}}} = \sum_{k=0}^{n} mdeg{~\sigma_{k}} = (s_{1},0,\dots,0) + (s_{2}, s_{2}, 0, \dots, 0) + \dots + (s_{n}, s_{n}, \dots, s_{n})$
($m\deg{~\sigma_{k}} = \underbrace{(1, 1,\dots, 1)}{k}, \underbrace{(0, \dots,0)}{n-k}$ и далее по лемме)

$\begin{cases}a_{1} = s_{1} + s_{2} + \dots s_{n} & s_{1} = a_{1} - a_{2} \geq 0 \ a_{2} = s_{2} + \dots s_{n} & s_{2} = a_{2} - a_{3} \geq 0 \ \dots\dots\dots\dots\dots\dots... & \dots\dots\dots\dots\dots \ a_{n - 1} = s_{n - 1} + s_{n} & s_{n - 1} = a_{n - 1} - a_{n} \geq 0 \ a_{n} = s_{n}\end{cases}$
$$g = f - A \sigma_{1}^{a_{1}} \sigma_{2}^{a_{2}} \dots \sigma_{n}^{a_{n}}$$
$$mdeg{~g} < mdeg{~f}$$ $g$ - симметрический степени меньше, чем наименьший, который не выражается через $\sigma_{k}$ $\Rightarrow$
$f$ - выражается через $\sigma_{k}$ противоречие.
$\square$
