Гипербола

Определение:


$$
\boxed{\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1}
$$
где $a > 0$, $b > 0$
- Две симметричные ветви при $|x| \geq a$
- Асимптоты: $y = \pm \frac{b}{a}x$
- Эксцентриситет: $e = \frac{c}{a} > 1$, где $c = \sqrt{a^2 + b^2}$

Фокусы:
Расположены на действительной оси:
$$
F_1(-c, 0), \quad F_2(c, 0)
$$

Директрисы:
Прямые, параллельные мнимой оси:
$$
x = \pm \frac{a}{e} = \pm \frac{a^2}{c}
$$

Асимптоты

$$y = \pm \frac{b}{a}x$$


Д-во:

$$ y(x) = kx + l $$ Выразим $y(x)$ для правой ветви (для левой просто будет с минусом): $$ y(x) = \frac{b}{a}\sqrt{x^2-a^2} $$ $$ k = \lim_{x\to+\infty} \frac{y(x)}{x} = \frac{b}{a} $$ $$ l = \lim_{x\to+\infty} y(x)-kx = \frac{b}{a}\lim_{x\to+\infty} \frac{x^2 - a^2 - x^2}{\sqrt{x^2-a^2}} = 0 $$

Лемма о фокальных расстояниях

Формулировка

Для точки $M(x,y)$ на правой ветви ($x > 0$): $$ \boxed{ \begin{aligned} MF_1 &= e x + a \\ MF_2 &= e x - a \end{aligned} } $$ Для левой ветви ($x < 0$): $$ \boxed{ \begin{aligned} MF_1 &= - (e x + a) \\ MF_2 &= - (e x - a) \end{aligned} } $$

Д-во:

Для случая, если $M$ на правой ветке (для левой аналогично) $$MF_{1} = \sqrt{(x - c)^{2} + y^{2}} = \sqrt{x^{2} - 2cx + c^{2} + b^{2}\left( \dfrac{x^{2}}{a^{2}} - 1 \right)} = $$ $$\sqrt{\dfrac{x^{2}(a^{2}+b^{2})}{a^{2}} - 2aex + c^{2} - b^{2}} = \sqrt{x^{2}e^{2} - 2aex + a^{2}} = |ex - a|$$ так как $M$ на правой ветви модуль можно снять со знаком плюс. $\square$

Критерий принадлежности точки к гиперболе

Формулировка:

Точка $M$ принадлежит гиперболе $\iff$ $|MF_1 - MF_2| = 2a$

Д-во:

$\Rightarrow$ из леммы $\Leftarrow$ $M(x, y)~|\sqrt{(x - c)^{2} + y ^{2}} - \sqrt{(x+c)^{2} + y^{2}}| = 2a$ $$\sqrt{(x - c)^{2} + y^{2}} - \sqrt{(x + c)^{2} + y^{2}} = \pm 2a$$ $$\sqrt{(x - c)^{2} + y^{2}} = \pm 2a + \sqrt{(x + c)^{2} + y^{2}} ~|\uparrow2$$ $$x^{2} - 2cx + c^{2} + y^{2} = 4a^{2} \pm 4a \sqrt{(x + c)^{2} + y^{2}} + x^{2} + 2cx + c^{2} + y^{2} |\mathpunct{:}~4$$ $$\pm a \sqrt{(x + c)^{2} + y^{2}} = a^{2} + cx | \uparrow 2$$ $$a^{2}x^{2} + 2a^{2}cx + a^{2}c^{2} +a^{2}y^{2} = a^{4} + 2a^{2}cx + c^{2}x^{2}$$ $$(c^{2} - a^{2})x^{2} - a^{2}y^{2} = a^{2}(c^{2}-a^{2})$$ $c^{2} - a^{2} = b^{2}$ $$b^{2}x^{2} - a^{2}y^{2} = a^{2}b^{2} |\mathpunct{:}~a^{2}b^{2}$$ $$\dfrac{x^{2}}{a^{2}} - \dfrac{y^{2}}{b^{2}} = 1$$ $\square$
