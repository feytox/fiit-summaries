Корневые подпространства

Определение: Нильпотентный оператор

Оператор $\mathcal{A}:V \to V$ называется нильпотентным, если $\exists{s}~~$такое, что $\mathcal{A}^{s}$ - нулевой оператор. Минимальное такое $s$ называется ступенью нильпотентности.

Определение: 1 и 0 компоненты оператора

Формулировка:

$U_{k} = \mathrm{Im}~\mathcal{A}^{k}$ - называется 1-компонентной $\mathcal{A}$ $V_{k} = \mathrm{Ker}~\mathcal{A}^{k}$ - называется 0-компонентной $\mathcal{A}$ где $k$ - ступень стабилизации.


Построение:


$$\begin{align}U_{1} &= {u \mid \mathcal{A}u \in \mathrm{Im}~\mathcal{A}} \\ U_{2} &= {u \mid \mathcal{A}^{2}u \in \mathrm{Im}~\mathcal{A}^{2}} \\ &\dots \\ U_{k} &= {u \mid \mathcal{A}^{k}u \in \mathrm{Im}~\mathcal{A}^{k}}\end{align}$$
$$U_{1}\supseteq U_{2} \supseteq \dots \supseteq U_{k}~~(1)$$

$$\begin{align}V_{1} &= {v \mid \mathcal{A}v \in \mathrm{Im}~\mathcal{A}} \\ V_{2} &= {v \mid \mathcal{A}^{2}v \in \mathrm{Im}~\mathcal{A}^{2}} \\ &\dots \\ V_{k} &= {v \mid \mathcal{A}^{k}v \in \mathrm{Im}~\mathcal{A}^{k}}\end{align}$$
$$V_{1} \subseteq V_{2} \subseteq \dots \subseteq V_{k}~~(1)$$

$(1)$ Eсли какие-то вектора передут в $0$, то они в нем и останутся для всех последующих степеней операторов: $$\mathcal{A}^{i}x = 0 ~\Rightarrow~\mathcal{A}^{i+1}x = \mathcal{A}(\mathcal{A}^{i}x) = \mathcal{A}(0) = 0$$
Таким образом, если в какой-то момент $V_{k-1} = V_{k}$, равенство будет выполнятся для всех последующих $V_{j},~j\geq k - 1$. Аналогично для $U_{k}$. Причем из теоремы о сумме ранга и дефекта для подпространств $V_{k}$ и $U_{k}$ это произойдет для одной и той же степени оператора.
По определению ядра и образа эти два подпространства будут $\mathcal{A}^{k}$ - инвариантны.

Разложение Фиттинга

Формулировка:

Для любого оператора $\mathcal{A}$ конечномерного пространства $V$ существуют $N_{\mathcal{A}}$ и $U_{\mathcal{A}}$ такие, что 1) $V = N_{\mathcal{A}} \oplus U_{\mathcal{A}}$ 2) $N_{\mathcal{A}}$ и $U_{\mathcal{A}}$ - $\mathcal{A}$ - инвариантны 3) $\mathcal{A}|_{N_{\mathcal{A}}}$ - нильпотентный оператор 4) $\mathcal{A}|_{U_{\mathcal{A}}}$ - обратимый оператор


Д-во:

В качестве $U_{\mathcal{A}}$ и $N_{\mathcal{A}}$ возьмем 1 и 0 компоненты оператора $\mathcal{A}$ соответственно. 1) $\mathrm{dim}~N_{\mathcal{A}} + \mathrm{dim}~ U_{\mathcal{A}} = \mathrm{dim}~V$ из их построения. Поэтому для прямой суммы достаточно показать, что $N_{\mathcal{A}}~\cap~U_{\mathcal{A}} = 0$ $x \in N_{\mathcal{A}}~\Rightarrow~\mathcal{A}^{k}x = 0$ $\exists{y} \in V~~x = \mathcal{A}^{k}y~ \Rightarrow \mathcal{A}^{2k}y = 0~\Rightarrow y \in V_{2k}$ $N_{\mathcal{A}}=V_{k} = \dots =V_{2k}~\Rightarrow y \in N_{\mathcal{A}}~\Rightarrow x = \mathcal{A}^{k}y = 0$ 2) Очевидно из построения компонент 3) $N_{\mathcal{A}} = V_{k} = \{x~|~\mathcal{A}^{k}x = 0 \}$ $\mathcal{A}^{k}|_{N_{\mathcal{A}}} = 0 = (\mathcal{A}|_{N_{\mathcal{A}}})^{k}$ т.e. оператор нильпотентен с степенью нильпотентности равной $k$ 4) $\exists{x}~\in~U_{\mathcal{A}}\mathpunct{:}~~\mathcal{A}x = 0,~x \neq 0$ $\exists{y}~\in~V\mathpunct{:}~~x = \mathcal{A}^{k}y \Rightarrow \mathcal{A}^{k+1} y = 0$, но $V_{k} = V_{k + 1} \Rightarrow x = \mathcal{A}^{k}y = 0$ $\square$

Определение: корневое подпространство

Формулировка:

Пусть $\lambda$ - собственное значение оператора $\mathcal{A}$, тогда корневым подпространством называется 0-компонента оператора $\mathcal{A} - \lambda E$.

Наблюдение: о корневом подпространстве

$U \subseteq V$, $U$ - $\mathcal{A}$ - инвариантно $\iff$ $U$ - $(\mathcal{A} - \lambda E)$ - инвариантно (следует и замкнутости пространства относительно умножения на скаляр) $(1)$ Из разложения Фиттинга и $(1)$ $\Rightarrow$ $$V = N_{\mathcal{A} - \lambda E} \oplus U_{\mathcal{A} - \lambda E}$$

Корневое разложение

Формулировка:

$\mathcal{A}$ - оператор пространства $V$, у которого характеристический многочлен раскладывается на линейные сомножители. Тогда: $$V = \mathop{\bigoplus}\limits_{i=1}^{k} K_{\mathcal{A}}(\lambda_{i})$$ где $\lambda_{i}~~~i=1\dots k$ - собственные числа $\mathcal{A}$ $K_{\mathcal{A}}(\lambda_{i})$ - корневое подпространство, размерность которого соответсвует кратности корня характерестического многочлена.


Д-во:


Индукция по $n = \mathrm{dim}~V$
База индукции: очевидна.
Шаг индукции: Рассмотрим $\lambda_{1}$ и $K(\lambda_{1})$
$$V = K_{\mathcal{A}}(\lambda_{1}) + V^{'}$$
$$A = \left(\begin{array}{c|c} B & 0 \\ \hline 0 & C \\\end{array}\right),~~B = [A|{K(\lambda{1})}],~~C = [A|{V^{'}}]$$
$$\chi{A}(\lambda) = \begin{vmatrix}B - \lambda E & 0 \\ 0 & C - \lambda E\end{vmatrix} = (B - \lambda E)(C - \lambda E) = \chi_{A|{K(\lambda{1})}}(\lambda) \cdot \chi_{A|_{V^{'}}}(\lambda)$$

Из разложения Фиттинга $\mathcal{A} - \lambda_{1}E$ обратим на $V^{{'}}~\Rightarrow~ \lambda_{1}$ не является собственным для $\mathcal{A}|{V^{'}}$
Пусть $\mu$ собственное для $\mathcal{A}|{K(\lambda_{1})}$.
Из нильпотентности $(\mathcal{A} - \lambda_{1})^{k}v = 0,~v \neq 0~\Rightarrow~(B - \lambda E) = (-1)^{k}(\lambda - \lambda_{1})^{k}$
Размерность $V^{{'}}$ меньше $V$ и в $V^{'}$ нет векторов отвечающих $\lambda_{1} ~\Rightarrow$ по предположению индукции $\square$
