Формулировка:
```spoiler-markdown
Если $F(x)$ - [[первообразная]] $f(x)$ на $X$, то множеством всех первообразных является:
$$\{F(x) + C \mid C \in \mathbb{R}\}$$
```

Д-во:
```spoiler-markdown
$\Large\implies$
$$F_{1}(x) = F(x) + C \implies F_{1}'(x) = F'(x) = f(x)$$

$\Large\impliedby$
Пусть $F_{1}(x)$ и $F_{2}(x)$ - первообразные $f(x)$, тогда:
$$(F_{1}(x) - F_{2}(x))' = f(x) - f(x) = 0$$
Тогда по [[Теоремы Ролля, Лагранжа и Коши о среднем. Следствия теоремы Лагранжа.#Следствие 1|следствию из теоремы Лагранжа]]:
$$F_{1}(x) - F_{2}(x) = C ~~~~~\square$$
```
