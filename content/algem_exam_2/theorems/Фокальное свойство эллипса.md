## Лемма
Формулировка:
```spoiler-markdown
Точка $M$ лежит на эллипсе, если $MF_{1} = a - ex$ и $MF_{2} = a + ex$
```
Д-во:
```spoiler-markdown
Посчитаем квадрат расстояния от $F_{1}$ до $M$
$$MF_{1}^{2} = (x-c)^{2} + y^{2} = $$
так как точка принадлежит эллипсу, то $y^{2} = b^{2}(1-\dfrac{x^{2}}{a^{2}})$
$$= x^{2} - 2xc + c^{2} + b^{2} - \dfrac{b^{2}}{a^{2}}x^{2} = \dfrac{a^{2} - b^{2}}{a^{2}}x^{2} - 2ea + a^{2} =$$
$$= e^{2}x^{2} - 2ea + a^{2 } = (a - ex)^{2}$$
$MF_{1} = |a - ex|$ т.к. $x$ принадлежит эллипсу, то модуль можно раскрыть с $+$ ($e < 1$ и $x \leq a$).
Для $F_{2}$ аналогично.
$\square$
```

## Фокальное свойство эллипса
Формулировка:
```spoiler-markdown
Точка $M$ лежит на эллипсе, если$$MF_{1} + MF_{2} = 2a$$
```
Д-во:
```spoiler-markdown
$\Rightarrow$ по лемме 
$\Leftarrow$ $MF_{1} + MF_{2} = 2a$ 
$$\sqrt{(x - c)^{2} + y^{2}}+ \sqrt{(x + c)^{2} + y^{2}} = 2a| \uparrow 2$$
$$x^{2} - 2xc + с^{2} + y ^{2} = 4a^{2} - 4a\sqrt{(x + c)^{2} + y^{2}} + x^{2} + 2xc + c^{2} + y^{2} |\mathpunct{:}4$$
$$a\sqrt{(x + c)^{2} + y^{2}} = xc + a^{2}|\uparrow 2$$
$$a^{2}(x^{2} + 2cx + c^{2} + y^{2}) = x^{2}c^{2} + 2xca^{2} + a^{4}$$
$$(a^{2}-c^{2})x^{2} + a^{2}y^{2} = a^{4} - a^{2}c^{2}$$
$$b^{2}x^{2} + a^{2}y^{2} = a^{2}b^{2}|\mathpunct{:}a^{2}b^{2}$$
$$\dfrac{x^{2}}{a^{2}} + \dfrac{y^{2}}{b^{2}} = 1$$
$\square$
```
