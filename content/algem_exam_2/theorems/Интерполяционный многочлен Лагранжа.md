Формулировка:
```spoiler-markdown
Пусть заданы различные элементы $a_0, a_1, \ldots, a_n$ бесконечного поля $F$ и произвольные элементы $b_0, b_1, \ldots, b_n \in F$. Тогда существует единственный многочлен $f(x) \in F[x]$ степени $\deg f(x) \leq n$, такой что:  
$$f(a_i) = b_i \quad \forall{i = 0, 1, \ldots, n}~~.$$
```
Д-во:
```spoiler-markdown
**Существование**
- Построим такой многочлен для частного случая.
Пусть $b_{0} = b_{1} = \dots = b_{i-1} = b_{i+1} = \dots = b_{n} = 0$
Построим многочлен который принимает значение $0$ в этих точках
$f(x) = (x-a_{0})(x-a_{1})\dots(x - a_{i-1})(x-a_{i+1})\dots(x - a_{n})$

Теперь добавим ненулевое значение
$f(a_{i}) = b_{i} \neq 0$
Подберем так константу $A$ чтобы значение в $a_i$ было $b_i$
$f(x) = A(x-a_{0})(x-a_{1})\dots(x - a_{i-1})(x-a_{i+1})\dots(x - a_{n})$
$f(a_{i}) = b_{i} = A(a_{i} - a_{0})(a_{i} - a_{1})\dots(a_{i} - a_{i-1})(a_{i} - a_{i+1})\dots(a_{i} - a_{n})$
$A = \dfrac{b_{i}}{\prod_{j \neq i}^{n}(a_{i} - a_{j})}$
Значит, $f_{j}(x) = \dfrac{b_{i} \prod_{j \neq i}^{n}(x - a_{j})}{\prod_{j \neq i}^{n}(a_{i} - a_{j})}~~~~~~~~\deg f_{i}(x) = n$
- Из частного $\to$ общий
Построим многочлен как сумму многочленов для частных случаев
Понятно что каждое слагаемое из всего выбора $a_i$ будет ненулевым 1 раз и тогда в каждой точке все слагаемые кроме 1 будут 0 и значения будут совпадать
$f(x) = f_{0}(x) + f_{1}(x) + \dots + f_{i}(x) + \dots + f_{n}(x)$
$f(a_{0}) = b_{0}$
$f(a_{1}) = b_{1}$
$\dots\dots\dots$
$f(a_{i}) = b_{i}$
$\dots\dots\dots$
$f(a_{n}) = b_{n}$

$\deg f(x) \leq n$ (так как степени не могут увеличиваться, но могут уменьшаться) $~~~~\square$

**Единственность**
о/п:
$$f_1(x_i) = f_2(x_i) = y_i$$
$$ g(x) := f_1(x) - f_2(x)$$
Степень $g$ не больше чем $n - 1$, но
$$ g(x_i) = f_1(x_i) - f_2(x_i) = 0$$
То есть он имеет n корней - противоречие
$\square$
```
