## Лемма 3: Дважды сопряжённое отображение
Формулировка:
```spoiler-markdown
Если для линейного оператора $f: V \to W$ существует сопряжённый ${} f^{*}\mathpunct{:}~~ W \to V {}$, то $(f^{*})^{*}\mathpunct{:}~~ V \to W$ существует и $(f^{*})^{*} = f$.
```

Д-во:
```spoiler-markdown
По определению $(f^{*})^{*}$, для всех $y \in W, x \in V$: $(f^{*}(y), x) = (y, (f^{*})^{*}(x))$.
По определению $f^*$: $(x, f^{*}(y)) = (f(x), y)$.
Используя свойство скалярного произведения $(u,v) = \overline{(v,u)}$, получаем 
$$(f^{*}(y), x) = \overline{(x, f^{*}(y))} = \overline{(f(x),y)} = (y, f(x))$$
Сравнивая два выражения для $(f^{*}(y), x)$, имеем $(y, (f^{*})^{*}(x)) = (y, f(x))$.
В силу невырожденности скалярного произведения, $(f^{*})^{*}(x) = f(x)$ для всех $x \in V$, т.е. $(f^{*})^{*} = f$.
$\square$
```

## Теорема о существовании сопряжённого оператора в конечномерных пространствах
Формулировка:
```spoiler-markdown
Пусть $V, W$ — конечномерные пространства над $F \in \{\mathbb{R}, \mathbb{C}\}$ со скалярным произведением. Оператор $f: V \to W$ имеет сопряжённый оператор $f^* \iff f$ линеен.
```

Д-во:
```spoiler-markdown
$\Large\implies$
Если $f^*$ существует, то $f^*$ линеен (Лемма 2). Тогда $(f^*)^*$ существует и $(f^*)^* = f$ (Лемма 3). Применяя Лемму 2 к $f^*$, его сопряжённый $(f^*)^* = f$ также линеен.

$\Large\impliedby$
Пусть $f$ линеен. Выберем ортонормированные базисы (ОНБ) $e$ в $V$ и $g$ в $W$. Пусть $A$ – матрица $f$ в этих ОНБ. Координаты векторов $x,y$ обозначим $x_c, y_c$. Тогда:
$$(f(x), y) = (Ax_c)^T \overline{y_c} = x_c^T A^T \overline{y_c}$$
Ищем $f^*: W \to V$ с матрицей $B$ (в базисах $g, e$) так, что $(x, f^*(y)) = x_c^T \overline{By_c}$.
Приравнивая: $x_c^T A^T \overline{y_c} = x_c^T \overline{By_c}$. Отсюда $A^T \overline{y_c} = \overline{By_c}$.
Комплексно сопрягая: $\overline{A^T} y_c = By_c$. Это верно для всех $y_c$, значит $B = \overline{A^T}$.
Обозначим $A^* = \overline{A^T}$ (эрмитово сопряжённая матрица). Оператор $f^*$ с матрицей $A^*$ в данных ОНБ является искомым сопряжённым оператором. Для $F=\mathbb{R}$, $A^* = A^T$.
$\square$
```

## Свойства эрмитова сопряжения матриц
Формулировка:
```spoiler-markdown
Для матриц $A, B$ и скаляра $\lambda$:
1. $(A + B)^{*} = A^{*} + B^{*}$
2. $(\lambda A)^{*} = \overline{\lambda} A^{*}$
3. $(AB)^{*} = B^{*} A^{*}$
4. $(A^{*})^{*} = A$
```

Д-во:
```spoiler-markdown
Используется определение $X^* = \overline{X^T}$ и свойства транспонирования и комплексного сопряжения.
$$(A + B)^{*} = \overline{(A+B)^T} = \overline{A^T + B^T} = \overline{A^T} + \overline{B^T} = A^* + B^*$$
$$(\lambda A)^{*} = \overline{(\lambda A)^T} = \overline{\lambda A^T} = \overline{\lambda} \overline{A^T} = \overline{\lambda} A^*$$
$$(AB)^{*} = \overline{(AB)^T} = \overline{B^T A^T} = \overline{B^T} \overline{A^T} = B^* A^*$$
$$(A^{*})^{*} = \overline{(A^*)^T} = \overline{(\overline{A^T})^T} = \overline{\overline{(A^T)^T}} = (A^T)^T = A$$
(Здесь использовано $\overline{(\overline{M})^T} = M$ и $(\overline{M})^T=\overline{M^T}$)
$\square$
```
