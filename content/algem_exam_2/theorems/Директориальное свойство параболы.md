Формулировка:
```spoiler-markdown
Точка $M$ лежит на параболе $\Leftrightarrow$ $d(M, l) = MF$, где $l$ - [[Парабола#Директриса|директриса параболы]], $F$ - [[Парабола#Фокус|фокус]].
```
Д-во:
```spoiler-markdown
$\Rightarrow$
$M(x, y),~y^{2} = 2px$
$$MF = \sqrt{(x - \dfrac{p}{2})^{2} + y^{2}} = \sqrt{x^{2} - px + \dfrac{p^{2}}{4} + 2px} = \sqrt{(x + \dfrac{p}{2})^{2}} = |x + \dfrac{p}{2}| = x + \dfrac{p}{2}$$
$\Leftarrow$
$MF = d(M, l)$
$$\sqrt{(x - \dfrac{p}{2})^{2} + y^{2}} = |x +\dfrac{p}{2}|$$
$$x^{2} - px + \dfrac{p^{2}}{4} + y^{2} = x^{2} + px + \dfrac{p^{2}}{4}$$
$$y^{2} = 2px$$
$\square$
```
