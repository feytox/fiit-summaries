## Тождества поглощения
Формулировка:
```spoiler-markdown
Для любых $a, b \in A$:
$$a \land (a \lor b) = a$$
$$a \lor (a \land b) = a$$
```

Д-во:
```spoiler-markdown
Докажем первое тождество: $a \land (a \lor b) = a$.
Для любого $x \in A$:
$$x \preceq a \implies x \preceq a \lor b \text{ по транзитивности}$$
Отсюда следует, что:
$$\bot((\{a, a \lor b\})) = \bot((\{a\}))$$
Тогда по определению инфимума:
$$a \land (a \lor b) = \inf(\{a, a \lor b\}) = \inf(\{a\}) = a$$
$\square$
```
