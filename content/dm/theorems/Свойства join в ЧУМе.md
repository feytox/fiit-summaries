## Теорема об операции $\lor$ в ЧУМе
Формулировка:
```spoiler-markdown
Операция $\lor$ в ЧУМе ассоциативна, коммутативна и идемпотентна.
```

Д-во:
```spoiler-markdown
**Идемпотентность:** $a \vee a = a$
Очевидно, т.к. $\{a, a\} = \{a\}$ и $\sup(\{a\}) = a$.

**Коммутативность:** $a \vee b = b \vee a$ 
Очевидно, т.к. $\{a, b\} = \{b, a\}$.

**Ассоциативность:** $(a \vee b) \vee c = a \vee (b \vee c)$
Заметим, что $(a \preceq x \text{ и } b \preceq x) \iff \sup(\{a, b\}) \preceq x$.
$x \in T(\{\sup(\{a, b\}), c\})$ означает, что $\sup(\{a, b\}) \preceq x \text{ и } c \preceq x$.

Используя предыдущее замечание, получаем следующую цепочку эквивалентностей:
$$x \in T(\{\sup(\{a, b\}), c\}) \iff a \preceq x \text{ и } b \preceq x \text{ и } c \preceq x \iff x \in T(\{a, b, c\})$$

Из этого следует, что $T(\{\sup(\{a, b\}), c\}) = T(\{a, b, c\})$.
Аналогично, $T(\{a, \sup(\{b, c\})\} ) = T(\{a, b, c\})$.
Следовательно, $T(\{\sup(\{a, b\}), c\}) = T(\{a, \sup(\{b, c\})\})$
Равенство множеств верхних границ означает равенство их наименьших верхних границ (супремумов):
$$\sup(\{\sup(\{a, b\}), c\}) = \sup(\{a, \sup(\{b, c\})\})$$
По определению операции $\lor$ как $\sup$, это эквивалентно $(a \vee b) \vee c = a \vee (b \vee c)$.
$\square$
```

## Следствие
```spoiler-markdown
$(A, \vee)$ — полугруппа.
Полурешетка = коммутативная полугруппа идемпотентов.
```
