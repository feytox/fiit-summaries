## 2-связный граф
Определение:
```spoiler-markdown
Граф (с более чем 2 вершинами) без точек сочленения называется **2-связным**.
```

## Свойство 2-связного графа
Формулировка:
```spoiler-markdown
Граф является 2-связным, если при удалении любой его вершины он остается связным.
```

## Компонента 2-связности
Определение:
```spoiler-markdown
**Компонентой 2-связности** (**2-связной компонентой**) графа называется любой его максимальный 2-связный подграф.
```

## Блоки графа
Определение:
```spoiler-markdown
**Блоками графа** называются его 2-связные компоненты и мосты.
```
