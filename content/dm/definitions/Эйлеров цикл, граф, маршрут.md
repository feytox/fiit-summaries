## Эйлеров цикл и граф
Определение:
```spoiler-markdown
Цикл в графе $G$ называется **эйлеровым**, если он в точности один раз проходит по каждому ребру графа.
Граф, у которого есть эйлеров цикл, называется **эйлеровым**.
```

```spoiler-markdown
Маршрут в графе $G$ называется **эйлеровым**, если он в точности один раз проходит по каждому ребру графа.
Граф, у которого есть эйлеров маршрут, называется **полуэйлеровым**.
```
