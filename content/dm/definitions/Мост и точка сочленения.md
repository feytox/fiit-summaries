## Мост
Определение:
```spoiler-markdown
Ребро $e$ является **мостом** в графе $G$ если при его удалении (ребра $e$) увеличивается число компонент связности.
```

## Точка сочленения
Определение:
```spoiler-markdown
Вершина $v$ — **точка сочленения** в графе, если при ее удалении увеличивается число компонент связности.
```

## Утверждение о концах моста
Формулировка:
```spoiler-markdown
По меньшей мере один из концов моста — точка сочленения (если это не 2-вершинный граф).
```
