## Решетка замкнутых классов
Определение:
```spoiler-markdown
$\mathcal{P}_{k}$ — решетка замкнутых классов функций на $k$-элементном множестве. 
Для случая $k=2$ (булевы функции) решетку называют решеткой Поста и обозначают $\mathcal{P}_{2}$
```

## Нуль и единица решетки Поста
Определение:
```spoiler-markdown
Единицей решетки $\mathcal{P}_{2}$ является класс всех булевых функций $\mathbf{B}$
Нулем решетки $\mathcal{P}_{2}$ является класс всех проекций $\mathbf{Pr} = \{PROJ_{i}\}$, состоящий из функций, задаваемых формулами без операторов или схемами без вентилей
```

## Атом и коатом решетки
Определение:
```spoiler-markdown
Элемент решетки называется **атомом**, если он покрывает $0$
Элемент решетки называется **коатомом**, если его покрывает $1$
```
