## Полная система
Определение:
```spoiler-markdown
Множество $B$ булевых функций называется **полной системой**, если формулой с множеством операций $B$ можно задать **любую** булеву функцию
```

## Примеры полных систем
Формулировка:
```spoiler-markdown
Примеры:

1) $K_1 = \{x \vee y, x \wedge y, \bar{x}\}$
Через $K_1$ получается любая ДНФ $\Rightarrow$ любая булева функция выражается через функции класса 1   
   
2) $K_2 = \{x \vee y, \bar{x}\}$
3) $K_3 = \{\downarrow\}$
4) $K_4 = \{'\}$
```
