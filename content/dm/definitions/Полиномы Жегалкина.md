## Поле $\mathbb{F}_2$
Определение:
```spoiler-markdown
**Поле** $\mathbb{F}_2$ — это множество $\{0, 1\}$ с операциями $+$ (по $\text{mod}~2$) и $\cdot~ (= \land)$
```

## Полином Жегалкина
Определение:
```spoiler-markdown
**Полином Жегалкина** — это многочлен от нескольких переменных (булевых) над полем $\mathbb{F}_{2}$
$$f(x_{1},x_{2},\dots, x_{n}) = \sum_{\vec{c} \in \{0,1\}^{n}} a_{\vec{c}}x_{1}^{c_{1}}x_{2}^{c_{2}}\dots x_{n}^{c_{n}}$$
```

## Свойства $\mathbb{F}_{2}$
Свойства:
```spoiler-markdown
- $x^{n} = x$
- $x + x = 0$
- $xy = x \land y$, $\bar{x} = x + 1 \implies \{+, \cdot, 1\}$ - полная система
```
